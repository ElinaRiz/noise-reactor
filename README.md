# NoiseReactor

Приложение для контроля шумов


## Описание

NoiseControl — это мобильное приложение, предназначенное для мониторинга уровня шума в помещении или на улице. Подходит для использования в офисах, учебных заведениях, студиях, а также в личных целях.

## Основные возможности

- Отслеживание уровня шума в реальном времени
- Уведомление при превышении допустимого порога
- Логирование данных для последующего анализа
